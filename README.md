# yeelight-cmd
cmd utility for my yeelight lamp

## Télécharger et lancer le script
### Windows 
```ps 
iex (irm "https://raw.githubusercontent.com/Hoferlukaslh/yeelight-bash/refs/heads/main/Install-Yeelight.ps1")
. $PROFILE

```

### Linux
#### CURL
```bash
curl -O https://raw.githubusercontent.com/Hoferlukaslh/yeelight-bash/refs/heads/main/install_yee.sh && chmod +x install_yee.sh && ./install_yee.sh

```

#### WGET
```bash
wget https://raw.githubusercontent.com/Hoferlukaslh/yeelight-bash/refs/heads/main/install_yee.sh && chmod +x install_yee.sh && ./install_yee.sh

```
